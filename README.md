# DevOps CA2

Simple web app with deployment automation.

## Setup

```bash
bun install
bun run src/index.ts
```

## Deploy

```bash
./scripts/deploy.sh
```

## Structure

- `src/` - app code
- `k8s/` - kubernetes configs
- `ansible/` - server setup
- `scripts/` - deployment scripts
